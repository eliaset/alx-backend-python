# 0x03. Unittests and Integration Tests

## Project Overview

This project focuses on testing in Python using `unittest`, `parameterized`, and `unittest.mock`. It covers:

- **Unit tests**: Testing individual functions and methods in isolation.
- **Integration tests**: Testing interactions between multiple components.
- **Mocking**: Avoiding real HTTP, file, or database calls in tests.
- **Parameterized tests**: Testing functions with multiple input values.
- **Memoization**: Testing decorators and cached properties.
